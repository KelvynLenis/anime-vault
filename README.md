# Anime vault

![Anime Website](https://i.ibb.co/MG1nbqt/YT-Thumbnails-2.png)

Simple project to fetch animes from a API using NextJS with infinite scroll and animations as they apear.

## Stacks

- NextJS
- Tailwind
- Framer Motion
